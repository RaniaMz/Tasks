#How to run the code

##To Run Twitter-Application;
build docker image and run it using the following commands:
cd AppForTwitter
docker build -t twitterapp .
docker run -it twitterapp 

##To Run Sudoku-Application;
build docker image and run it using the following commands:
cd Sudoku
docker build -t sudoku .
docker run -it sudoku
